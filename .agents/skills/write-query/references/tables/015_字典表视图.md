## 常用 JOIN 模式

### 069 服务状态 `state` 中文名

```sql
attr_id = '4000000201'
AND attr_value = cast(state AS string)
AND city_id = '200'
-- 取 attr_value_name
```

### 产品规格 / 附属产品属性 `attr_value1` 中文名

```sql
a.attr_id = b.attr_id
AND a.attr_value1 = b.attr_inner_value
AND b.city_id = '200'
-- 取 b.attr_value_name
```

### 设备购买方式 / 设备来源中文名

```sql
a.property_type = b.attr_inner_value
AND b.attr_id = 4000000208
AND b.city_id = '200'
-- 取 b.attr_value_name
```

> `state` 用 **`attr_value`**；产品规格特性（105）与附属产品特性（106）均用 **`attr_inner_value`**。如果要翻译 `attr_id` 本身的特性名称，用 `016_字典维表视图.md` 的 `attr_name`。
