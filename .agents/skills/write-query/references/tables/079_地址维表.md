# 地址维表

- **Hive 表名**: `zone_gz_yz.dwd_yz_addr_final`
- **用途**: 通过主业务表中的地址标识补充中文地址名称，或由装机标准地址回填 7 级地址 ID/名称。

---

## 字段说明

| 字段 | 字段类型 | 字段含义 |
|------|---------|---------|
| id | decimal(24,0) | 地址id |
| addr | string | 地址 |
| grade | int | 级别 |
| parentid | decimal(24,0) | 上级 |
| addr_id_6 | decimal(24,0) | 6级地址id |
| addr_id_7 | decimal(24,0) | 7级地址id |

---

## 本表常用条件速查（A 层附注）

| 叫法/场景 | 条件片段 |
|----------|---------|
| 装机地址中文名 | `grade = 10` |
| 069 全业务资料表补地址 | `main.serv_addr_id = CAST(addr_dim.id AS STRING)` |
| 069 补 7 级地址 | `main.serv_addr_id = CAST(addr_dim.id AS STRING)` 后取 `addr_dim.addr_id_7`，再二次关联 `addr7.grade=7` |

> 地址 ID 类型提醒：069 `serv_addr_id` 是字符型，本表 `id/addr_id_7` 是 `decimal(24,0)`。关联时统一转字符，禁止默认 `CAST(main.serv_addr_id AS DECIMAL(24,0)) = addr_dim.id`，避免长地址 ID 或异常字符转换失败导致漏数。

## 常用 SQL 片段

```sql
LEFT JOIN (
    SELECT DISTINCT
        id,
        addr
    FROM zone_gz_yz.dwd_yz_addr_final
    WHERE grade = 10
) addr_dim
    ON main.serv_addr_id = CAST(addr_dim.id AS STRING)
```

## 7 级地址 SQL 片段

```sql
LEFT JOIN zone_gz_yz.dwd_yz_addr_final addr_dim
    ON main.serv_addr_id = CAST(addr_dim.id AS STRING)
LEFT JOIN zone_gz_yz.dwd_yz_addr_final addr7
    ON CAST(addr_dim.addr_id_7 AS STRING) = CAST(addr7.id AS STRING)
   AND addr7.grade = 7
```

## 地址脱敏模板

```sql
CASE
    WHEN LENGTH(addr_dim.addr) < 4 THEN addr_dim.addr
    WHEN LENGTH(addr_dim.addr) = 4 THEN CONCAT(SUBSTR(addr_dim.addr, 1, 1), '*')
    WHEN LENGTH(addr_dim.addr) > 4 THEN CONCAT(SUBSTR(addr_dim.addr, 1, LENGTH(addr_dim.addr) - 4), '****')
    ELSE NULL
END AS tm_addr_name
```
