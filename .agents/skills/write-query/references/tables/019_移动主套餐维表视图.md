# 移动主套餐维表视图

- **Hive 表名**: `metadata_ods_day.md_ft_cdma_disc_config`
- **用途**：按移动主套餐 ID 回填移动主套餐名称。
- **典型入口**：069 全业务资料表 `cdma_disc_type`，以及其它移动号码事实表中的 `cdma_disc_type`。
- **业务负责人**: 
- **业支负责人**: 杨洋

---

## 字段说明

| 字段 | 字段类型 | 字段含义 |
|------|---------|---------|
| cdma_disc_id |  | 移动主套餐 ID |
| cdma_disc_desc |  | 移动主套餐名称 |

## 常用 SQL

### 069 补移动主套餐名称

```sql
select
  a.serv_id,
  a.acc_nbr,
  a.cdma_disc_type,
  d.cdma_disc_desc
from dwm_yz_tb_comm_cm_all_final a
left join metadata_ods_day.md_ft_cdma_disc_config d
  on a.cdma_disc_type = d.cdma_disc_id
where a.par_month_id = ${month_id}
;
```

## 风险与审计

- 069 的 `cdma_disc_type` 是移动主套餐 ID；需要输出中文主套餐名称时补本表 `cdma_disc_desc`。
- 如果主表已自带稳定的主套餐名称字段（如 `cdma_disc_desc`），优先直接取主表字段，不重复补维表。
- 不要把移动主套餐维表和 020 销售品维表混用：移动主套餐名称走本表；销售品 / offer 编码名称走 020。
