# OpenClaw Skills 页面设计风格指南

> 提炼自 `OpenClaw Skills：让 AI 从「会说话」到「会做事」.html`，供其他 AI 工具参考生成同风格页面。

---

## 一、色彩体系

| 角色 | 色值 | 用途 |
|------|------|------|
| **主色（Brand）** | `#e11d48`（rose-600） | 按钮、强调文字、图标、边框高亮、渐变 |
| **主色浅** | `#fef2f2` → `#fca5a5`（brand-50~300） | 浅色背景块、badge 底色、装饰光晕 |
| **主色深** | `#be123c` → `#881337`（brand-700~900） | hover 态、深色强调 |
| **中性背景** | `#f8fafc`（slate-50）、`#ffffff`（white） | 页面底色、section 交替背景 |
| **中性文字** | `#1e293b`（slate-800）主文字、`#475569`（slate-600）副文字 | 正文与说明 |
| **深色底** | `#0f172a`（slate-900）、`#1e293b`（slate-800） | 代码块、终端窗口、深色 section |
| **成功绿** | `#10b981`（emerald-500） | 闭环图中的"完成"节点 |

**核心原则**：以品牌红为唯一彩色，其余全部用 slate 灰阶。页面最多出现 2 种彩色（红 + 绿点缀）。

---

## 二、排版体系

| 层级 | 规格 | 使用场景 |
|------|------|----------|
| **Hero 标题** | `text-5xl~7xl`（3rem~4.5rem）`font-black` | 页面主标题 |
| **Section 标题** | `text-4xl`（2.25rem）`font-bold` | 各 section 标题 |
| **卡片标题** | `text-xl`（1.25rem）`font-bold` | 卡片内标题 |
| **正文** | `text-base~lg`（1rem~1.125rem）`text-slate-600` | 段落文字 |
| **小字/标签** | `text-xs~sm`（0.75rem~0.875rem）`uppercase tracking-wider` | section 标签、badge |
| **代码** | `font-mono text-sm` | 代码片段、文件名 |

**字体**：`Inter`（无衬线正文）+ 系统等宽字体（代码）。中文与英文混排统一使用 Inter。

**特殊效果**：关键标题使用 `gradient-text`（红→浅红渐变文字）。

---

## 三、页面结构

```
┌─────────────────────────────────┐
│  Hero（全屏封面）                 │  ← 渐变背景 + 浮动光晕 + 居中 CTA
├─────────────────────────────────┤
│  Section 1（白色背景）            │  ← 居中标题 + 内容
├─────────────────────────────────┤
│  Section 2（slate-50 背景）       │  ← 交替背景色区分章节
├─────────────────────────────────┤
│  Section 3（白色背景）            │
├─────────────────────────────────┤
│  ... 交替 ...                   │
├─────────────────────────────────┤
│  深色 Section（slate-800/900）   │  ← 演示/总结区域用深色
└─────────────────────────────────┘
```

**每个 Section 的标准头部**：

```html
<div class="text-center mb-16">
    <span class="text-brand-600 font-semibold tracking-wider text-sm uppercase">英文标签</span>
    <h2 class="text-4xl font-bold text-slate-800 mt-2 mb-4">中文标题</h2>
    <p class="text-slate-600 text-lg">一句话副标题</p>
    <div class="w-20 h-1 bg-brand-500 mx-auto rounded-full mt-4"></div>
</div>
```

**关键特征**：英文标签 + 中文标题 + 副标题 + 红色短横线装饰。

## 四、核心组件模式

### 1. 卡片（Card）

```css
/* 白色卡片 */
bg-white p-8 rounded-2xl shadow-sm border border-slate-200

/* 品牌色卡片 */
bg-brand-50 p-8 rounded-2xl shadow-sm border border-brand-200

/* hover 效果 */
transition: all 0.3s ease;
hover: transform: translateY(-5px);
       box-shadow: 0 20px 40px -10px rgba(225, 29, 72, 0.15);
       border-color: rgba(225, 29, 72, 0.3);
```

### 2. 终端/代码窗口（Mac Window）

```
┌──────────────────────────┐
│ ●  ●  ●  my-skill/      │  ← 深色标题栏 + 三色圆点
├──────────────────────────┤
│  monospace 代码内容       │  ← 深色背景 + 浅色等宽字
└──────────────────────────┘
```

- 背景：`#0f172a`，标题栏：`#1e293b`
- 圆点：红 `#ff5f56` / 黄 `#ffbd2e` / 绿 `#27c93f`
- 代码高亮：关键字用品牌色 `#f87171`

### 3. 对话气泡

```
用户消息：右对齐，品牌色背景，白色文字，圆角梯形
AI 回复：  左对齐，白色背景，灰色边框，圆角梯形
```

### 4. 时间线/步骤条

```
● ──── 卡片内容
● ──── 卡片内容      ← 左侧圆形编号 + 竖线 + 右侧卡片
● ──── 卡片内容
```

- 圆形编号：品牌色渐变填充，带阴影
- 竖线：`bg-gradient-to-b from-brand-600 to-brand-300`

### 5. 流程图节点（深色底）

```
[开始] ──→ [查找] ──→ [分支] ──→ [汇聚] ──→ [完成]
```

- 节点用圆角矩形，品牌色/深色交替
- 连线用细竖线/横线
- 完成节点用 emerald 绿色发光效果

## 五、装饰元素

| 元素 | 实现 |
|------|------|
| **背景光晕** | `absolute` 定位大圆，`blur-3xl` + `opacity-20` + `mix-blend-multiply` |
| **Badge 标签** | 圆角 `rounded-full`，品牌色浅底 + 品牌色文字，带图标 |
| **装饰线** | `w-20 h-1 bg-brand-500 rounded-full` 居中 |
| **图标** | Font Awesome 6，品牌色或 slate-400 |
| **右侧导航点** | `fixed` 定位，圆点，当前 section 高亮放大 |

---

## 六、动画规范

| 动画 | 用途 | 参数 |
|------|------|------|
| `fade-in` | 元素入场 | 0.8s ease-out，opacity 0→1 |
| `slide-up` | 标题入场 | 0.6s ease-out，translateY(30px)→0 + opacity |
| `pulse-slow` | 背景光晕呼吸 | 3s infinite |
| `hover-lift` | 卡片悬停 | 0.3s ease，上移 5px + 阴影增强 |
| `section-fade` | 滚动可见 | IntersectionObserver，threshold 0.1 |

---

## 七、布局约束

- 内容区：`max-w-4xl`、`max-w-5xl`、`max-w-6xl` 三档，居中 `mx-auto`
- 内边距：section 上下 `py-24`，容器 `px-6`
- 卡片间距：`gap-6` 或 `gap-8`
- 响应式：`grid-cols-1 md:grid-cols-2 lg:grid-cols-3`

---

## 八、技术栈

- **Tailwind CSS**：通过 CDN 引入 `https://cdn.tailwindcss.com`
- **字体**：Google Fonts `Inter`
- **图标**：Font Awesome 6 CDN
- **动画**：纯 CSS + IntersectionObserver（无额外 JS 库）

## 九、设计原则总结（可直接放入 Prompt）

> **你是一个擅长制作技术产品介绍页面的 UI 设计师。请按以下风格生成一个单页 HTML：**
>
> 1. **色彩**：以 `#e11d48`（玫瑰红）为唯一品牌色，其余全部用 slate 灰阶（slate-50/100/200/400/600/800/900）。白色和浅灰交替作为 section 背景。深色 section 用 slate-900。
> 2. **字体**：Inter 无衬线字体，标题用 bold/black 粗细，正文用 slate-600。关键标题用红→浅红渐变文字。
> 3. **布局**：全屏 Hero 封面 → 多个 section 交替（白/灰底）→ 深色收尾。每个 section 居中 max-w-4xl~6xl，头部统一：英文小标签 + 中文大标题 + 副标题 + 红色短横线装饰。
> 4. **卡片**：大圆角（rounded-2xl/3xl），浅阴影，hover 时上移 5px + 品牌色阴影。品牌色卡片用 brand-50 背景 + brand-200 边框。
> 5. **代码块**：Mac 风格三色圆点终端窗口，深色背景 `#0f172a`。
> 6. **动画**：滚动渐显（IntersectionObserver），入场 fade-in/slide-up，背景光晕 pulse。
> 7. **装饰**：绝对定位的大圆光晕（blur-3xl + 低透明度），Font Awesome 图标，右侧固定导航圆点。
> 8. **对话气泡**：用户右对齐品牌色底，AI 左对齐白底灰边框。
> 9. **时间线**：左侧圆形编号 + 竖线 + 右侧卡片。
> 10. **整体气质**：专业、现代、有呼吸感，像 Apple 风格的产品介绍页。克制用色，大量留白，信息密度适中。
