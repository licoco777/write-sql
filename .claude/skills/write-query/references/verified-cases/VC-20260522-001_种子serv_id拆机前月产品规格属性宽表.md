# 种子 serv_id 拆机前月产品规格属性宽表

来源：write-query 交互验证（接入号种子表 + 逻辑拆机月 + 特性月表 + 宽表）

## 适用

- 用户提供 **`serv_id` 种子清单表**（可有 `acc_nbr`，**无拆机月**），要对每个服务打标。
- 取 **拆机前一个月** 的 **产品规格属性**（`attr_id` → `attr_value1` + 中文名）。
- 多个 `attr_id`，输出 **宽表**（每个 attr 两列：`attr_{id}_val` / `attr_{id}_name`）。
- 拆机口径未特说明时，默认 **逻辑拆机**。

## 不适用

- 用户明确 **物理拆机** 且有拆机订单事实 → 改 `is_wl_cancel_user=1` + `wl_cancel_subs_stat_date`。
- 取 **当前在网** 特性 → 特性**日表** `summary_ods_day_city.tb_pre_cm_attr_all`。
- 销售品在档 / 订购动作 → 014 / 041，不是特性资料表。
- 固定账期打标（种子已带拆机月且不需反查）→ 可简化 Step1，但仍用特性月表。

## 主表与口径

- **驱动表**：用户种子表（如 `tmp_acc_nbr_list`），键 `serv_id`。
- **拆机月定位**：069 **月表** `dwm_yz_tb_comm_cm_all_mon_final`；**逻辑拆机** `is_cancel_user=1`；`cancel_month_id = par_month_id`（拆机标签仅拆机当月分区）；时间 `hist_create_date`；多次拆机取最近。
- **属性月**：`attr_month_id = cancel_month_id - 1`（`add_months(hist_create_date, -1)`）。
- **特性快照**：105 特性资料**月表** `iodata_ods_month_city.tb_pre_cm_attr_all_mon`；`par_month_id=attr_month_id` + `par_corp_id='200'` + `attr_id IN (...)`。
- **特性中文**：`dws_crm_cfguse.dws_attr_value`；`attr_value1 = attr_inner_value` + `city_id='200'` + `attr_id` 对齐。

## 字段映射

| 需求字段 | 来源 |
|----------|------|
| acc_nbr, serv_id | 种子表 |
| hist_create_date, cancel_month_id, attr_month_id | 069 月表 + 计算 |
| attr_{id}_val | 特性月表 `attr_value1` |
| attr_{id}_name | 字典 `attr_value_name` |

## 补表

| 缺口 | 补表 | JOIN 键 |
|------|------|---------|
| 拆机月 | 069 月表 | `serv_id` + `is_cancel_user=1` |
| 拆机前月特性 | 105 特性月表 | `serv_id` + `par_month_id=attr_month_id` + `attr_id` |
| 特性中文 | 015 字典表 | `attr_id` + `attr_value1=attr_inner_value` + `city_id='200'` |

## SQL 编排

CTAS 流水线（ORC + Snappy）；种子驱动全程 LEFT JOIN。

```sql
-- Step1：种子 + 逻辑拆机月（069 月表）
drop table if exists tmp_ads_yz_cj_attr_cancel_base purge;
create table tmp_ads_yz_cj_attr_cancel_base
row format delimited fields terminated by '\u0001'
stored as orc tblproperties('orc.compression'='snappy') as
select
    s.acc_nbr,
    s.serv_id,
    cm.hist_create_date,
    cm.par_month_id as cancel_month_id,
    cast(date_format(add_months(cm.hist_create_date, -1), 'yyyyMM') as int) as attr_month_id
from tmp_acc_nbr_list s
left join (
    select serv_id, par_month_id, hist_create_date,
           row_number() over (partition by serv_id order by hist_create_date desc, par_month_id desc) as rn
    from dwm_yz_tb_comm_cm_all_mon_final
    where is_cancel_user = 1
) cm on s.serv_id = cm.serv_id and cm.rn = 1;

-- Step2：拆机前一月特性长表
drop table if exists tmp_ads_yz_cj_attr_snap_base purge;
create table tmp_ads_yz_cj_attr_snap_base
row format delimited fields terminated by '\u0001'
stored as orc tblproperties('orc.compression'='snappy') as
select b.serv_id, b.attr_month_id, a.attr_id, a.attr_value1
from tmp_ads_yz_cj_attr_cancel_base b
left join iodata_ods_month_city.tb_pre_cm_attr_all_mon a
    on b.serv_id = a.serv_id
   and b.attr_month_id = a.par_month_id
   and a.par_corp_id = '200'
   and a.attr_id in ('500040209', '500040210');

-- Step3：宽表 + 中文
drop table if exists ads_yz_cj_attr_pre_cancel_wide purge;
create table ads_yz_cj_attr_pre_cancel_wide
row format delimited fields terminated by '\u0001'
stored as orc tblproperties('orc.compression'='snappy') as
select
    b.acc_nbr, b.serv_id, b.hist_create_date, b.cancel_month_id, b.attr_month_id,
    max(case when s.attr_id = '500040209' then s.attr_value1 end) as attr_500040209_val,
    max(case when s.attr_id = '500040209' then d209.attr_value_name end) as attr_500040209_name,
    max(case when s.attr_id = '500040210' then s.attr_value1 end) as attr_500040210_val,
    max(case when s.attr_id = '500040210' then d210.attr_value_name end) as attr_500040210_name
from tmp_ads_yz_cj_attr_cancel_base b
left join tmp_ads_yz_cj_attr_snap_base s on b.serv_id = s.serv_id
left join dws_crm_cfguse.dws_attr_value d209
    on d209.attr_id = '500040209' and d209.city_id = '200'
   and d209.attr_inner_value = s.attr_value1 and s.attr_id = '500040209'
left join dws_crm_cfguse.dws_attr_value d210
    on d210.attr_id = '500040210' and d210.city_id = '200'
   and d210.attr_inner_value = s.attr_value1 and s.attr_id = '500040210'
group by b.acc_nbr, b.serv_id, b.hist_create_date, b.cancel_month_id, b.attr_month_id;
```

### 注意

- 069 回溯场景用 **月表**，不用日表 `dwm_yz_tb_comm_cm_all_final`。
- 特性历史用 **月表**；日表 `tb_pre_cm_attr_all` 只在网。
- 新增 `attr_id`：Step2 扩展 `IN (...)`，Step3 增加一对 `max(case...)` 列及字典 JOIN。
- Step1 建议加 `par_month_id` 范围裁剪以控性能。

### 过程表血缘

| 步骤 | 表名 | 用途 | 验收 |
|------|------|------|------|
| 1 | `tmp_ads_yz_cj_attr_cancel_base` | 拆机月/属性月 | `select count(*) ...` |
| 2 | `tmp_ads_yz_cj_attr_snap_base` | 特性长表 | `select attr_id,count(*) group by attr_id` |
| 3 | `ads_yz_cj_attr_pre_cancel_wide` | 宽表结果 | `select count(*) ...` |
