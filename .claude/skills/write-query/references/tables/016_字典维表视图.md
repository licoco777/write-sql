## 字段说明

| 字段 | 字段类型 | 字段含义 |
|------|---------|---------|
| attr_id | string | 特性标识 |
| attr_inner_cd | string | 特性编码 |
| attr_name | string | 特性名称 |

---

## 常用用途

| 需求 | JOIN / 过滤 | 输出 |
|------|-------------|------|
| 根据 `attr_id` 翻译特性名称 | `a.attr_id = spec.attr_id` | `spec.attr_name` |
| 根据产品规格编码 / 附属产品编码反查特性 | `spec.attr_inner_cd IN (...)` | `attr_id, attr_name` |

```sql
select attr_id, attr_name
from dws_crm_cfguse.dws_attr_spec
where attr_inner_cd in (...)
```

---

## 与特性值字典的边界

- 本表回答“这个 `attr_id` / `attr_inner_cd` 是什么特性”，输出 `attr_name`。
- `015_字典表视图.md` 回答“这个 `attr_value1` 码值是什么意思”，输出 `attr_value_name`。
- 产品规格属性（105）和附属产品属性（106）都可用本表补 `attr_name`。
