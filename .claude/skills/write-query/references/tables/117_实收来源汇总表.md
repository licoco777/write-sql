# 实收来源汇总表

- **Hive 表名**: `zone_gz_yz.dwd_yz_if_real_src_sum_new_final`
- **生产表名**: `zone_gz_yz.dwd_yz_if_real_src_sum_new_final`
- **业务含义**: 号码 / 服务级实收来源汇总表，用于按号码、接入号或服务清单查询实收金额。
- **粒度**: 号码 / 服务 / 月实收来源明细，可按 `par_month_id, serv_id, acc_nbr` 汇总到月度实收。
- **分区字段**: `par_month_id`

---

## 常用场景

- 实收取数完整流程（附件清单、业务条件圈定、直查汇总、种子驱动）见 `scenarios/SC-009_047117收入实收查询.md`。
- 按附件号码或接入号清单查询指定账期范围的实收金额。
- 按月打横输出实收金额，例如 `SUM(CASE WHEN par_month_id=YYYYMM THEN ss_amount ELSE 0 END)`。
- VPN / 专线项目清单查实收时，若 VPN / 专线范围由附件保证，可直接按 `acc_nbr` 匹配本表；若用户要求校验产品属性，再补 069 或 033。

---

## 核心字段

| 字段 | 字段类型 | 字段含义 |
|------|----------|----------|
| par_month_id | string / int | 账期月份 |
| serv_id | string / decimal | 服务标识 |
| acc_nbr | string | 接入号 / 号码 |
| flag | string | 实收来源类型，常见 `HF`、`OT` |
| amount | numeric | 金额 |
| amount_tc | numeric | 套餐 / 调整抵扣金额，`HF` 口径需扣减 |

---

## 实收金额口径

```sql
sum(case when flag = 'HF' then amount - amount_tc else 0 end)
+ sum(case when flag = 'OT' then amount else 0 end) as ss_amount
```

使用建议：

- 先按 `par_month_id, serv_id, acc_nbr` 汇总实收，再按输出维度或月份打横。
- 多月取数必须写闭区间，例如 `par_month_id BETWEEN ${start_month} AND ${end_month}`。
- 附件号码清单按 `acc_nbr` 匹配；需要保持附件顺序时保留附件序号。

---

## 注意事项

- 本表承接“实收金额 / 实收来源”场景，不等同于 047 最终版划小收入、048 全量科目级收入、097 基本面月清单。
- 不要用 069 全业务资料表的费用字段替代实收来源口径。
- 附件或标题中的 VPN / 专线项目范围不等于本表自动校验了产品属性；需要校验时另补产品资料表。
