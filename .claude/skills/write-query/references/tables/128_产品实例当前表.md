# 产品实例当前表

- **Hive 表名**：`dws_crm_cust.dws_prod_inst`
- **业务定位**：CRM 产品实例当前资料表，用于按服务实例或接入号回填当前 CRM 原始报装地址。
- **数据范围**：只看当前数据，没有历史月表。
- **常用城市过滤**：广州固定 `city_id=200`。

---

## 关键字段

| 字段 | 字段类型 | 字段含义 | 说明 |
|------|----------|----------|------|
| prod_inst_id | string/number | 产品实例 ID | 可转换为 069/主表口径中的 `serv_id` |
| acc_num | string | 接入号 / 号码 | 可作为 `acc_nbr` 使用 |
| address_desc | string | 报装地址 | CRM 原始报装地址 |
| city_id | string/number | 地市 | 广州 `200` |

---

## 常用 SQL

```sql
select
  cast(prod_inst_id as decimal(22,0)) as serv_id,
  acc_num as acc_nbr,
  address_desc
from dws_crm_cust.dws_prod_inst
where city_id = 200
```

主表按服务实例补报装地址：

```sql
left join (
  select
    cast(prod_inst_id as decimal(22,0)) as serv_id,
    acc_num,
    address_desc
  from dws_crm_cust.dws_prod_inst
  where city_id = 200
) prod_inst
  on cast(base.serv_id as decimal(22,0)) = prod_inst.serv_id
```

---

## 使用提醒

- 本表的 `address_desc` 是 CRM 原始报装地址，不等同于 079 地址维表的标准装机地址。
- 用户要求“标准装机地址”“装机地址层级”“几级地址”时，仍走 069 `serv_addr_id` + 079 地址维表。
- 本表只存当前数据，不能用于历史账期地址回溯。
- 源案例里的 `decimai(22,0)` 是拼写错误，生成 SQL 时应修正为 `decimal(22,0)`。
